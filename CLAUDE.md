# Reading Challenge 项目

## 项目简介
阅读挑战计划：在 2026/4/18 - 6/16（59天）内完成中英文各 20 本书的阅读目标，共 40 本。
读者为五年级男生，兴趣方向：F1 赛车、科普、冒险。

## 项目结构

```
reading-challenge/
├── CLAUDE.md           # 项目规则
├── PROGRESS.md         # 进度日志
├── README.md           # 书单总览 + 进度追踪
├── 阅读计划.md          # 详细 8 周计划 + 购买清单
├── 英文书单.md          # 英文候选书单（原始版）
├── 中文书单.md          # 中文候选书单（原始版）
├── DESIGN.md           # 设计文档
└── *.heic / *.jpeg     # 书籍照片（用于识别已有书目）
```

## 关键文件说明
- `README.md`：最终确定的 40 本书单（中英文各 20 本），带 checkbox 追踪完成状态
- `阅读计划.md`：按周排列的阅读计划，包含购买优先级和关键节点检查
- `英文书单.md` / `中文书单.md`：原始候选书单，书目多于最终选定数量

## 工作流程
1. 更新阅读进度时：在 `README.md` 中勾选对应书目，在 `阅读计划.md` 中更新关键节点状态
2. 调整计划时：修改 `阅读计划.md` 的周计划部分
3. 每次更新后同步 `PROGRESS.md`
