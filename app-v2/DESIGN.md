# 阅读挑战 App v2 — 设计文档

## 设计理念
Duolingo 风格，色彩丰富活泼，适合五年级学生。每本书直接显示进度，打卡零摩擦。

## 页面结构

### 底部 Tab 导航（2个）
- 📚 **中文书** — 中文书列表
- 📖 **英文书** — 英文书列表

### 顶部统计栏（两个 Tab 页共享）
固定在页面顶部，显示核心数据摘要：
- 🔥 连续打卡天数
- 📅 剩余天数（倒计时）
- 📊 总进度（如 5/40 本）

点击顶部统计栏 → 进入 **详情页面**，显示：
- 中文/英文各自的完成进度环形图
- 打卡日历热力图
- 每日阅读页数趋势
- 已读/在读/未读统计

---

## 主页面（中文书 / 英文书）

### 书籍卡片列表
每本书是一个卡片，纵向排列，包含：
- 📕 书名 + 作者
- 进度条（彩色渐变，显示百分比）
- 当前页数 / 总页数
- 状态标识：未开始 / 阅读中 / 已完成
- 未拥有的书用灰色/虚线边框区分

### 点击书籍卡片 → 弹窗打卡
弹窗内容：
- 书名
- 当前进度条
- 输入框："读到第___页"（默认显示当前页数）
- 确认按钮
- 自动计算：本次阅读页数 = 新页数 - 旧页数
- 读完自动标记完成 🎉

### 右上角管理入口
点击进入 **管理页面**：
- 拖拽排序书籍顺序
- 添加新书
- 编辑书籍信息（页数、作者等）
- 删除书籍
- 标记是否已拥有

## 子页面
- **详情页**：统计详情，从顶部栏点进去，有返回按钮
- **管理页**：书籍排序/增删改，从右上角⚙️进去，有返回按钮
- **打卡弹窗**：覆盖层弹窗，点击书籍卡片触发

## 配色方案
- 主色：多彩渐变（紫→蓝→绿→橙），不同书用不同颜色
- 背景：浅色（米白/淡灰）
- 进度条：彩色渐变
- 卡片：白底 + 大圆角 + 柔和阴影
- 已完成：金色/绿色庆祝效果
- 未拥有：灰色调

## 技术栈
- React 18 + Vite + TailwindCSS v4
- Supabase（复用现有数据库）
- Netlify 部署
